# Personal Website

This website is created for IAT355 Assignment 2: Personal Website.

The purpose of this website is to present my background, skills, projects, and interests in a professional online format. It also includes interactive data visualizations created using JavaScript and SVG.


## Technologies Used

- HTML  
- CSS  
- JavaScript  
- SVG  
- GitHub Pages  

## Website Navigation

This website contains two main pages:

- Home (`index.html`)  
  Includes my personal introduction, education, skills, and project section.

- Visualizations (`visualizations.html`)  
  Displays two interactive SVG visualizations created using JavaScript.

The navigation bar at the top of the page allows users to switch between different sections and pages easily.
